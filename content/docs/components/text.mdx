## Installation

```bash
npm install motion
```

## Usage

```tsx
import Text_01 from "@/components/xui/text/text-01";

export function MyComponent() {
  return (
    <div className="w-full max-w-md">
      <Text_01 />
    </div>
  );
}
```

## Props

Each text variant has its own specific props. Refer to the individual component implementations for detailed prop information.

## Dependencies

This component requires the following packages:

- `motion` (framer-motion)
